# Solution Notebooks

This folder contains complete solution versions of the Python Basics notebooks.
The directory structure mirrors the main learning path so that each solution can
be compared directly with the corresponding exercise notebook.

Use the main notebooks first. Then open the matching notebook in this folder to
compare your approach with a worked solution and read the explanatory comments.

## Folder Map

```text
solution/
|-- 1_Getting_Started/
|-- 2_Basics/
|-- 3_Data_Structures/
`-- 4_Functions/
```

## How to Use the Solutions

1. Complete the exercise in the main notebook.
2. Open the matching solution notebook from this folder.
3. Compare the structure of your code with the worked solution.
4. Re-run the solution cells if you want to inspect the output step by step.

The solution notebooks are intentionally runnable from top to bottom. Intentional
error examples from the main notebooks remain commented out or are replaced by
safe explanations so the solution path stays executable.
